# taceo-nodes-observability

[![Crates.io](https://img.shields.io/crates/v/taceo-nodes-observability.svg)](https://crates.io/crates/taceo-nodes-observability)
[![docs.rs](https://docs.rs/taceo-nodes-observability/badge.svg)](https://docs.rs/taceo-nodes-observability)
[![License: MIT OR Apache-2.0](https://img.shields.io/badge/license-MIT%20OR%20Apache--2.0-blue.svg)](https://github.com/TaceoLabs/nodes-helpers/blob/main/LICENSE-MIT)

Setup helpers for metrics, logging and tracing for nodes in our MPC networks.

Configures tracing (via `telemetry-batteries`) and metrics with pluggable backends: Datadog, StatsD, or Prometheus (scrape or push).
