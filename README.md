# nodes-helpers

[![License: MIT OR Apache-2.0](https://img.shields.io/badge/license-MIT%20OR%20Apache--2.0-blue.svg)](https://github.com/TaceoLabs/nodes-helpers/blob/main/LICENSE-MIT)

Workspace with shared helper crates for TACEO MPC nodes.

## Crates

| Crate | Description |
|-------|-------------|
| [`taceo-nodes-common`](./nodes-common) | Common utilities: health API, PostgreSQL, Ethereum RPC, AWS secrets, graceful shutdown |
| [`taceo-nodes-observability`](./nodes-observability) | Telemetry setup: tracing, logging, and metrics (Datadog, StatsD, Prometheus) |
